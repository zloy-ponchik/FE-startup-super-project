# FE-startup-super-project
The frontend part of an innovative startup project

Как установить и протестировать:

    Откройте Chrome и перейдите по адресу chrome://extensions/.
    Включите “Режим разработчика” (Developer mode) в правом верхнем углу.
    Нажмите кнопку “Загрузить распакованное расширение” (Load unpacked) и выберите папку с вашим расширением.
    !!! Файл manifest.json должен находиться в корневой директории
    Теперь, перейдя на любую веб-страницу вы можете пользоваться расширением.
